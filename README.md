# Titanic Data Analysis (EDA)

A Python-based Exploratory Data Analysis (EDA) on the classic Titanic dataset to uncover key demographic trends and insights that influenced passenger survival rates.

## 🚀 Project Overview
This project focuses on cleaning, visualizing, and analyzing passenger data from the Titanic disaster. It explores how factors like socioeconomic status (class), gender, age, and family relationships impacted survival outcomes.

## 📊 Key Highlights & Analysis
* **Data Cleaning:** Handled missing values across key features like `Age` and `Embarked`.
* **Exploratory Visualizations:** Generated statistical plots using Seaborn and Matplotlib to analyze survival distributions.
* **Feature Relationships:** Examined critical correlations, such as the impact of Passenger Class (`Pclass`) and `Sex` on survival rate.

## 🛠️ Tech Stack & Dependencies
* **Language:** Python
* **Libraries:** `pandas`, `numpy`, `matplotlib`, `seaborn`
